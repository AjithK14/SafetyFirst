    How to use this Web App

    This is a web app designed to appropriately redistrict Emergency Medical Services (EMS). This app will take the amount of EMS available, all data from past crimes, and recommend where EMS should be placed to facilitate fast treatment.

    In the future more file formats will definitely become available, but for now in order for this app to properly make recommendations, a CSV file is required. The CSV must have the latitude and longitude coordinates of each crime. The number of EMS available must be manually entered. The app will take this data from past crimes, predict where future crimes will happen, and recommend where the existing stations should be placed.

    The app will download (Longitude, Latitude) coordinates for each precinct to your computer. The purpose of the map is to visualize and gain a contextual understanding on how the precincts were spaced out. To best use it, zoom in on the reddest areas, or areas with highest crime.
